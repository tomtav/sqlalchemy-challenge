Percipitation by Date

GET /api/v1.0/precipitation

Returns a collection of precipitation data for the last year in the database.

All Stations

GET /api/v1.0/stations

Returns a collection of all the stations in the database.

Temperatures from Most Active Station

GET /api/v1.0/tobs

Returns a collection of temperatures for the most active station (USC00519281) for the last year of data.
